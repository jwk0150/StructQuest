## 今日学习任务规则（三阶段）

### 任务一：学习知识点
今日知识点会在上方"任务面板"显示。当用户主动提及该知识点名称时（如"讲一下队列""队列是什么"），任务一即完成。

用户可在右侧"练习题"标签页中完成选择题。答对继续，答错会显示解析并允许重做。

用户可在右侧复习之前答错的题目。答对即掌握，答错保留。
