# 图

## 学习目标

理解图的基本术语、存储结构和遍历方法；掌握最小生成树、最短路径、拓扑排序和关键路径等经典算法。

## 基本概念

图由顶点集和边集组成，可分为有向图、无向图、带权图、连通图、强连通图等。顶点的度、入度、出度、路径、回路和连通分量是描述图结构的基本术语。

邻接矩阵用二维数组表示顶点关系，适合稠密图，空间复杂度为 `O(V^2)`。邻接表为每个顶点维护边链表，适合稀疏图，空间复杂度为 `O(V + E)`。

## 遍历

DFS 沿一条路径尽可能深入，再回溯，通常用递归或栈实现。BFS 按层扩展，通常用队列实现；在无权图中，BFS 首次到达某顶点就是最短路径。

## 经典算法

最小生成树用于在连通带权无向图中选择总权值最小的连接方案。Prim 从点集合扩展，适合稠密图；Kruskal 按边权排序，配合并查集，适合稀疏图。

Dijkstra 求非负权图的单源最短路径，Floyd 求所有顶点对最短路径。拓扑排序用于 DAG，可检测有向图是否有环。关键路径用于 AOE 网，决定工程最短完成时间。

## 易错点

DFS 和 BFS 的访问顺序可能因邻接表顺序不同而不同。Dijkstra 不能直接处理负权边。拓扑排序只适用于有向无环图。
