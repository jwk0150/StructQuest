# 串、数组和广义表

## 学习目标

掌握串的基本操作和模式匹配思想；理解多维数组的存储地址计算；掌握稀疏矩阵压缩存储和广义表的递归定义。

## 串

串是由零个或多个字符组成的有限序列。串的基本操作包括求长、复制、比较、连接、求子串和定位。模式匹配是在主串中寻找模式串出现位置。

BF 算法直接回退比较，最坏复杂度较高。KMP 算法利用模式串自身的前后缀信息构造 `next` 数组，失配时避免主串指针回退，时间复杂度为 `O(n + m)`。

## 数组

数组是类型相同元素的集合，通常顺序存储。二维数组按行优先或列优先存储，地址计算依赖基地址、元素大小和下标偏移。

## 稀疏矩阵

当矩阵中非零元素远少于零元素时，可使用三元组 `(row, col, value)` 只保存非零项。十字链表适合行列方向都需要高效遍历和修改的场景。

## 广义表

广义表是线性表的推广，元素可以是原子，也可以是另一个广义表。它适合表达递归结构。

## 易错点

KMP 的关键不只是背 `next` 数组公式，而是理解失配后模式串能右移多少。稀疏矩阵压缩存储节省空间，但会牺牲某些随机访问效率。
